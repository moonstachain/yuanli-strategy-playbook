## A2 · 元工具底盘卡：OSA + MECE + 信息五层萃取

用法：动手做任何战略任务前，先过这张卡。

#### A2.1　OSA 卡（动手前先填，不跳层）

> ━━━━━ OSA 卡 ━━━━━ O 目标层：要做什么？为什么是这件不是那件？（全局最优，≥3 候选对比） S 策略层：怎么打？≥3 候选方案比成功率/资源（信息五层萃取） A 行动层：谁做、何时做、怎么验真？（SMART 五字母齐） ✅ 完成标志：_______　📓 进化记忆：_______

> 核心铁律 必须先有完整 O 才能进 S，必须先有完整 S 才能进 A。跳层 = 80% 项目失败的根因。人天性跳到 A（动作有快感）——强制按 O→S→A 走，3 个月白干率降 70%。

#### A2.2　MECE 八层体检（验任何框架）

| 检验 | 问法 |
| --- | --- |
| 互斥 ME | 各项之间有没有重叠？（重叠=分类不干净） |
| 穷尽 CE | 各项加起来有没有遗漏？（遗漏=有盲区） |
| 不可再分 | 顶层能不能再约简到更少的类？ |
| 同一维度 | 各项是不是同一个分类维度（不能混维度） |

用法：本书每层框架（两不变量/四关/三链路/四维护城/产品阶梯）都过 MECE——**自创框架前先自检互斥+穷尽，再用。**

#### A2.3　信息五层萃取（S 段最容易翻车）

> 碎片信息 → 特征信息 → 观点信息 → 洞察信息 → 全局最优信息 （原始）   （归类）    （判断）    （规律）    （决策依据）

S 段不是「列 3 个方案」，本质是**认知从碎片爬到全局最优。**直接列方案没爬五层 = 拍脑袋，是战略最大的敌人。

#### A2.4　六判断扫描（任务启动前 1 分钟）

| 判断 | 问法 |
| --- | --- |
| 自治 | 信息够不够开工？ |
| 全局最优 | 这件事在全局里值不值得做？ |
| 能力复用 | 有现成 skill 可调吗？还是造新轮子？ |
| 验真 | 完成标准是什么？谁验？怎么验？ |
| 进化 | 做完留下什么可复用资产？ |
| 最大失真 | 我现在最容易在哪翻车？ |

口诀：**自治、全局、复用、验真、进化、失真。**扫 1 分钟救 1 晚白干，ROI ≈ 1:5000。

## A3 · 三链路速查：六要素 + 转化话术库 + 交付峰终

用法：写内容、谈转化、设计交付前，翻这张卡。

#### A3.1　获客 · 翻译六要素（所有内容统一规范）

| 要素 | 落地 |
| --- | --- |
| 极简 | 一眼看懂，不绕 |
| 权威 | 荣格/康波/段永平等真源背书 |
| 故事 | 康波故事 / 学员故事 / 自己故事 / AI 商科故事 |
| 细节 | 具体实操，不空泛 |
| 利他 | 金句 / 硬核长文 / 实操方法论，先给价值 |
| 便捷 | 一句入脑 / 一键生成，降行动成本 |

> 营销四阶段 + 渠道优先级 四阶段：认知 → 心智隔离+信任加工 → 体验（¥1.28万）→ 实战拿结果 渠道：创始人出镜短视频（最优）＞ 图文 ＞ 不做长期直播（太耗人）

#### A3.2　转化 · 增信话术库

| 场景 | 话术 / 心法 |
| --- | --- |
| 用户说「我都懂」 | 「这个不重要，你的东西很重要、很贵」——焦点从产品移到用户 |
| 降低决策压力 | 「精力不费力、时间不费力、钱不费力，你来体验怎么了？」 |
| 传递松弛 | 我们不缺这一单——松弛本身就是最强信任信号 |
| 不辩护价格 | 聚焦用户自身价值，不纠结产品价格（一辩护就掉进比价框架） |
| 讲事实不承诺 | 讲同结构学员拿到的真实结果，不讲「你也一定能赚」（伦理红线） |

#### A3.3　交付 · 惊艳速查

| 要点 | 落地 |
| --- | --- |
| 核心指标 | NPS（净推荐值）：流程标准化 + 超预期 |
| 峰终定律 | 刻意造一个「峰」（专家大脑震撼/现场出报告）+ 漂亮的「终」（汇报+点评） |
| 核心模式 | 用户自主完成关键战役 + 导师点评打分（不替用户做、不做前端技能） |
| 前置说清 | 获客转化阶段就讲清「教什么、不教什么」，避免预期落差 |
| 空档期 | 用户业务变→选新战役→持续用 OS+大脑→留在体系内 |

## A4 · 关键战役选型表：按生命周期选仗

用法：不知道当期该打哪一仗时，按所处阶段查可选战役。
关键战役的选择是有模型的，不是「今天想做啥做啥」。**按业务生命周期，给出该阶段撬动力最大的可选战役。**漏斗哪一层最窄，哪一层就是当期关键战役。

| 生命周期 | 该阶段的瓶颈 | 可选关键战役（选撬动全局那一个） |
| --- | --- | --- |
| 萌芽 · 找 PMF | 品类/甜用户没锁死 | 原力觉醒定型 / 品类独创验证 / 一句入脑测试（5 人听懂愿付） |
| 冷启动 · 获客窄 | 公域到私域转化不起来 | 创始人出镜短视频跑通 / 心智隔离词确立 / 落地页测试 |
| 成长 · 转化窄 | 私域成交率低 | 松弛转化 SOP / 瑞贝卡式增信话术标准化 / 体验款发酵升班 |
| 规模化 · 交付窄 | 交付不标准、依赖人 | 六天课前后分段 SOP / 自主完成+点评模式 / 原力大脑落地 |
| 守富 · 利润池 | 已有基本盘没挖透 | 利润池系统化（挖未开发空间）/ 升班复购机制 / 数据飞轮搭建 |
| 跃迁 · 天花板 | 规模化跑通后 | 专家大脑私有化 / FDE 企业版试点 / 讲师人才管线 |

> 选战役四步（配 A2 的 OSA） ① 全局最优：列 ≥3 个候选战役，按「对全局的撬动力」排序 ② 假设验证：每个战役建立在明确假设上，用最小代价（2万/4周）先验真伪 ③ SMART 锁定：具体/量化/可达/相关/限期，五字母齐 ④ 挂战略驾驶舱：进度实时可见，撬动全局那一仗集中打穿

> 2026 年三件锁定（示范） 当期最窄是获客——所以三件核心战役：① 创始人出镜短视频精准获客跑通；② 转化 SOP 标准化；③ 后三天交付标准化。三件做完，1000 万基础盘夯实，明年翻番。

## A5 · SABC / ABCS 双诊断

用法：给自己或学员做能量级 + 段位双轨自评。
两套诊断各管一件事：**SABC 量「能量级」（这门生意现在多强）；ABCS 量「段位」（操盘手成熟度）。**配合用，定位精准。

#### A5.1　SABC 能量级诊断（生意强度）

| 级 | 判据 | 对应动作 |
| --- | --- | --- |
| S 级 | 三标签甜用户、真实付费、口碑转介绍、四维护城成型 | 重仓投入、规模化 |
| A 级 | 有人真实付费、模式跑通一个真实订单（LTV/CAC>3） | 补壁垒、做留存 |
| B 级 | 有兴趣有咨询、但转化不稳、模式未闭环 | 回炉转化 SOP、验假设 |
| C 级 | 品类/甜用户未锁死、自嗨型 | 回第一~二册重做觉醒+品类 |

#### A5.2　ABCS 段位反弹（操盘手成熟度）

> 新手 vs 老手的根本差别 A 段（新手）：遇问题先动手（跳到 OSA 的 A），白干率高 B 段：会列方案，但不爬信息五层、不做全局最优对比 C 段：能 MECE 拆解、会选关键战役，但壁垒意识弱 S 段（老手）：反射性地先 O 后 S 再 A、框架过 MECE、战役挂驾驶舱、时刻想守富

#### A5.3　双诊断对照用法

| 组合 | 解读 / 处方 |
| --- | --- |
| 生意 S 级 + 操盘手 A 段 | 好生意撞上嫩手——补操盘能力（练 OSA/MECE），别糟蹋好牌 |
| 生意 C 级 + 操盘手 S 段 | 老手做错的事——回第一~二册重选品类/甜用户 |
| 生意 B 级 + 操盘手 B 段 | 双补：先用 A4 选当期关键战役，再用 A2 元工具跑流程 |
| 生意 S + 操盘手 S | 出师候选——进第六册出师机制，触发投资/跃迁 |

## A7 · 失真护栏 + 内容红线合规清单

用法：写任何对外内容、做任何交付前，最后过一遍这张卡。

#### A7.1　失真护栏（防止认知漂移）

| 护栏 | 自检问法 |
| --- | --- |
| 真源优先 | 我说的这条，是来自真源（逐字稿/底表），还是我自己编的？ |
| 抗套话漂移 | 我是不是滑回了 lean startup / 通用商科套话，丢了原力的反共识？ |
| 抗 AI 工具崇拜 | 我是不是把 AI 当主角了？（应是放大器，卖框架不卖工具） |
| 抗「我以为我懂」 | 这个概念我能讲清本体，还是只会背名词？ |
| 数据透明 | 引用的数据，标没标注「自报/未审计」、加没加泡沫风险提示？ |

#### A7.2　内容红线（伦理三红线，第六册第 3 章）
