# 原力战略 · 绿皮书

> **方法论正典 · 10x 深密版**
>
> 把「做自己」，变成「好生意」。

```
财富 = 借势 × 合力
```

整门生意只有两个不变量，且必须**相乘**。绿皮书全六部加一卷附录，都建立在这一行公式之上——先证明这门生意为什么成立（**道**），再锁定它是谁（**体**），再说战略怎么生产（**法**）、生意怎么跑（**术**）、东西怎么交付（**器**）、怎么守十年（**守**），最后给一卷随身工具卡（**附录**）。

## 🧭 怎么读

- **第一次通读** → 严格按 `道 → 体 → 法 → 术 → 器 → 守` 顺序。地基不稳，上面盖什么都会塌。
- **操盘时速查** → 直接翻 [第七册 · 附录卷](books/07-附录-appendix.md)，7 张工具卡是"查 / 调用，不是读"。
- **每一章统一结构**——「究竟完备五件套」，缺一件即判定不合格：

| 件 | 名称 | 回答的问题 |
|---|---|---|
| 件 1 | 本体定义 | 第一性是什么（不是形态，是本质） |
| 件 2 | 操作模板 | 怎么照着动手（可复制的模板 / 动作 / Prompt） |
| 件 3 | 合格性自检 | 什么算做对了（逐项可勾选） |
| 件 4 | 反本能 / 失败模式 | 人最容易在哪翻车（真实反例 + Q&A） |
| 件 5 | 真源引用 | 出处链接回课程转写 / 底表（防失真） |
